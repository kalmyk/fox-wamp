# FOX-WAMP Protocol APIs

This directory contains protocol-level API descriptions for the FOX-WAMP project. The files here describe the router-facing APIs for each supported protocol and the internal Hyper API used for inter-node communication and the router core.

Files:
- `wamp.md` — WAMP protocol mapping and supported messages/options
- `mqtt.md` — MQTT topic mapping and supported features
- `hyper.md` — Internal Hyper API (types, events, command semantics)

These documents are intended to be referenced from the main OpenSpec documentation so the API descriptions are discoverable by contributors and the OpenSpec toolchain.
